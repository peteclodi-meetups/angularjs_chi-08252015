E2E Testing: Manual vs Automated

Manual

  1. Launch your App yourself

  2. Perform your Testing

    • Possibly documented

    • All errors might be noticed

    • Tests might be performed the same each time

  3. Note Bugs/Create Bug Reports

  4. Fix bug(s) and update your codebase

  5. Hope you tested the system fully

    • But you might have missed something

  6. Return to Step 1

Automated

  1. Setup Test Environment

  2. Configure your test runner

  3. Write Tests

  4. Run Codified Tests

  5. Note Bugs/Create Bug Reports

  6. Fix bug(s) and update your codebase

  7. Know you tested the system fully

    • Because of Step 3!!!

  8. Return to Step 4
